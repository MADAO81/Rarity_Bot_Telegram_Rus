## 📖 О проекте

Рарити — это интерактивный бот, который:

- 💎 **Вдохновляет на творчество** — идеи для рукоделия, пэчворка, шитья, алмазной мозаики и картин по номерам.
- 👗 **Даёт советы по стилю** — сочетание цветов, модные тенденции, капсульный гардероб.
- 🪡 **Помогает с ремонтом одежды** — практичные советы, как зашить, подшить, пришить пуговицу.
- 🎨 **Подбирает цветовые гаммы** — гармоничные палитры для одежды и интерьера.
- ✨ **Вдохновляет** — цитаты о красоте, творчестве и гармонии.
- 💬 **Общается с характером Рарити** — великодушная, драматичная, эстетичная и немного театральная.

## 🛠️ Технологии

| Компонент | Технология |
|-----------|------------|
| Платформа | **Telegram** |
| ИИ-провайдер | **OpenAI GPT-4-turbo** |
| Распознавание голоса | **OpenAI Whisper** |
| Анализ изображений | **OpenAI Vision API** |
| Погода | **Open-Meteo** (бесплатно, без ключа) |
| База данных | **SQLite** |
| Язык | Python 3.11+ |
| Деплой | SprintBox (systemd) |

## 📋 Команды

| Команда | Описание |
|---------|----------|
| `/start` | Начать общение с Рарити 💎 |
| `/help` | Справка 📖 |
| `/style` | Совет по сочетанию цветов или модный тренд 👗 |
| `/craft` | Идея для рукоделия 🧵 |
| `/repair` | Советы по ремонту одежды 🪡 |
| `/inspire` | Вдохновляющая идея для творчества ✨ |
| `/subscribe` | Подписаться на ежедневные рассылки 📬 |
| `/unsubscribe` | Отписаться от рассылок 😢 |
| `/weather` | Погода 🌤️ |
| `/cleardata` | Очистить историю диалога 🗑️ |

## 📝 Ежедневные рассылки

* **9:30** — **«Бриллиантовое вдохновение»** — короткая фраза, цитата или идея, которая заряжает творчеством на весь день.
* **18:00** — **«Совет от Рарити»** — практичный совет по рукоделию, стилю или ремонту одежды.

> 🔔 Подписка оформляется через команду `/subscribe`.

## 📝 Особенности поведения

### 🎯 Реакция на сообщения в группах

| Ситуация | Поведение |
|----------|-----------|
| Упоминание бота (`@username`) | ✅ Всегда отвечает |
| Ответ на сообщение бота | ✅ Всегда отвечает |
| Обычное сообщение (без упоминания) | ❌ Не отвечает |
| Личное сообщение | ✅ Всегда отвечает |

### ⏰ Рабочее время
**Понедельник — Воскресенье: 9:00 — 22:00**

Вне рабочего времени: игнорирует сообщения в группах, в личке отвечает: *«Я отдыхаю, приходите завтра!»*

## 🚀 Быстрый старт

### 1. Клонируйте репозиторий

```bash
git clone https://github.com
cd Rarity_Bot_Telegram_Rus
```

### 2. Создайте виртуальное окружение

```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate     # Windows
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Настройте переменные окружения

Скопируйте файл `.env.example` в `.env` и заполните параметры:

```env
# Telegram
TELEGRAM_TOKEN=ваш_токен_бота

# OpenAI
OPENAI_API_KEY=ваш_ключ
OPENAI_MODEL=gpt-4-turbo

# Координаты Ворсино
DEFAULT_LAT=55.0965
DEFAULT_LON=36.6355

# Настройки
WORK_START_HOUR=9
WORK_END_HOUR=22
CONTEXT_EXPIRE_DAYS=30

# Администратор
ADMIN_ID=ваш_telegram_id

# Режим отладки
DEBUG_MODE=false
```

### 5. Запустите бота

```bash
python run.py
```
